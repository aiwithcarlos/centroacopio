# 🇻🇪 Centro Acopio Venezuela

**Aplicación web para registrar los centros de acopio alrededor del mundo que están recibiendo insumos médicos y alimentos para ayuda humanitaria a Venezuela**, tras los dos terremotos del 24 de junio de 2026.

<p align="center">
  <img src="public/favicon.svg" alt="Bandera de Venezuela" width="120" />
</p>

## 🚀 Stack Tecnológico

- **Frontend:** [Next.js 16](https://nextjs.org/) + [React 19](https://react.dev/) + [Tailwind CSS 4](https://tailwindcss.com/) + [TypeScript](https://www.typescriptlang.org/)
- **Backend:** API Routes de Next.js
- **Base de Datos:** [Supabase](https://supabase.com/) (PostgreSQL)
- **Mapas:** [Leaflet](https://leafletjs.com/) + OpenStreetMap
- **Despliegue:** [Vercel](https://vercel.com/) — [centroacopio.com](https://centroacopio.com)
